Sejam bem-vindos! Esta página é complementar ao curso de Hidrologia Estocástica, ofertado no Programa de Pós-Graduação em Engenharia de Recursos Hídricos e Ambiental da UFPR. Todas as implementações em linguagem R elaboradas para o material das aulas estão aqui contidas. Vocês encontram links para o download das aulas, das bases de dados utilizadas e dos códigos elaborados. Além deles, a página traz os cadernos explicativos dos códigos, para auxiliar no entendimento das lógicas utilizadas. Importante: página em constante atualização, conforme o curso avança.
